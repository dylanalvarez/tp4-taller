client -> server: opcode 0: nickname y id del equipo elegido
client -> server: opcode 1: nickname y id del mapa elegido

client -> server: opcode 2: elemento elegido

server -> client: opcode 3: nuevo mensaje de chat
client -> server: opcode 3: nuevo mensaje de chat

server -> client: opcode 2: estado del juego (cualquiera sean los datos que se dan periodicamente) (a definir)

client -> server: opcode 4: marcar casilla
client -> server: opcode 5: aplicar hechizo
client -> server: opcode 6: aplicar upgrade
client -> server: opcode 7: marcar casilla
